# Cloud Automation Lab

## Overview
This repository is a personal lab for learning and practicing Infrastructure as Code (IaC) and IT automation.

It includes experiments and learning materials related to:

- Azure cloud automation
- Ansible network automation
- Terraform infrastructure provisioning
- Microsoft 365 automation (PowerShell)
- AI-assisted development workflows

---

## Goals

The main goals of this project are:

- Build hands-on experience with cloud and infrastructure automation
- Practice real-world DevOps workflows using GitHub
- Learn how to manage infrastructure as code
- Explore AI-assisted coding (Codex / Claude / AI agents)

---

## Technologies

- GitHub (version control & collaboration)
- Terraform (Azure infrastructure provisioning)
- Ansible (configuration automation)
- PowerShell (Microsoft 365 management)
- Azure (cloud platform)

---

## Structure (future)

This repository will gradually include:

- `/terraform` → Azure infrastructure code
- `/ansible` → network and server automation playbooks
- `/powershell` → M365 automation scripts
- `/docs` → architecture notes and diagrams
